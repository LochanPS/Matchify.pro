# Matchify.pro Admin Code of Conduct

**Version 1.0** | Last Updated: December 28, 2025

---

## Purpose

Admins have elevated privileges to maintain platform integrity. With great power comes great responsibility.

---

## Core Principles

### 1. **Impartiality**
- Treat all users equally regardless of personal relationships
- Make decisions based on facts, not emotions
- Avoid conflicts of interest
- Never favor friends or family in disputes

### 2. **Transparency**
- All admin actions are logged (immutable audit trail)
- Provide clear reasons for suspensions/cancellations
- Document major decisions
- Be honest about mistakes

### 3. **Privacy & Security**
- Never share user data publicly
- Protect payment information
- Use secure passwords (2FA when available)
- Log out of shared devices
- Never access user accounts without consent

### 4. **Account Separation**
- Admins CANNOT play, organize, or umpire matches
- Create separate personal accounts for non-admin activities
- Never abuse admin privileges for personal gain
- Maintain clear boundaries between admin and personal roles

### 5. **Incident Response**
- Respond to critical issues within 1 hour
- Escalate payment disputes immediately
- Document all incidents in audit logs
- Follow established procedures

## Emergency Procedures

### User Suspension

1. **Verify violation** (screenshot/evidence)
2. **Document reason** in system (minimum 10 characters)
3. **Send email** to user explaining reason
4. **Set appropriate duration:**
   - Minor: 1-3 days
   - Moderate: 7-14 days
   - Serious: 30-90 days
   - Severe: 365 days
5. **Log action** in audit trail

### Tournament Cancellation

1. **Contact organizer first** (unless fraud/emergency)
2. **Verify refunds** will process correctly
3. **Document reason** (minimum 10 characters)
4. **Notify all participants** via email
5. **Process refunds** automatically
6. **Log action** in audit trail

### Data Breach Response

1. **Immediately revoke** all admin sessions
2. **Notify security team** at security@matchify.pro
3. **Force password reset** for affected users
4. **Document incident** in detail
5. **Post-mortem analysis** within 48 hours

### Payment Disputes

1. **Gather evidence** from both parties
2. **Review transaction logs**
3. **Contact payment gateway** if needed
4. **Make fair decision** based on evidence
5. **Document resolution**
6. **Process refund** if warranted

## User Suspension Guidelines

### Minor Violations (1-3 days)
- Spam in chat/comments
- Minor unsportsmanlike conduct
- First-time policy violations

### Moderate Violations (7-14 days)
- Repeated spam
- Harassment of other users
- Fake tournament registrations
- Multiple policy violations

### Serious Violations (30-90 days)
- Severe harassment
- Fraudulent activity
- Match fixing attempts
- Payment fraud

### Severe Violations (365 days or permanent)
- Identity theft
- Confirmed match fixing
- Repeated fraudulent activity
- Threats of violence
- Illegal activity
