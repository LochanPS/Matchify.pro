HOW TO TEST
===========================================

1. Start backend: cd backend && npm run dev
2. Start frontend: cd frontend && npm run dev
3. Navigate to: http://localhost:5173/scoring/{matchId}
4. Click "Start Match"
5. Observe timer starts
6. Click "Pause" - timer stops, scoring disabled
7. Click "Resume" - timer continues, scoring enabled
8. Score to 20-15 - game point alert appears
9. Complete first set
10. Score to 20-15 in set 2 - match point alert appears
11. For doubles: observe service rotation indicator
