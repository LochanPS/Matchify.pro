ROUTE ADDED:
------------

GET /matches/:matchId/live
- Displays detailed match view
- Real-time WebSocket updates
- Large scoreboard
- Match timeline

WEBSOCKET EVENTS:
-----------------

Client → Server:
- subscribe:match (matchId)
- unsubscribe:match (matchId)

Server → Client:
- match:scoreUpdate:${matchId} (score)
- match:statusChange:${matchId} (data)
