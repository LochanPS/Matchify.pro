{ "pointsPerSet": 5, "maxSets": 3, "setsToWin": 2, "extension": true }

{ "pointsPerSet": 11, "maxSets": 5, "setsToWin": 3, "extension": true }

2. Set 1: Score reaches 10-10 (deuce)
3. Player 1 scores: 11-10 (not won yet - need 2 point lead)
4. Player 1 scores: 12-10 (set won!)
5. Set completion modal: "Continue to Set 2?" or "End Match Here?"

üîß Configuration Options

Points Per Set

Any number: 2, 5, 11, 15, 21, 30, etc.

Configurable via +/- buttons or direct input

Number of Sets

1-9 sets supported

Single set or best-of format (e.g., best of 3, best of 5)

Extension (Deuce)

Yes: Must win by 2 points after reaching target

No: First to reach target points wins
